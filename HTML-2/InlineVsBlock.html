Inline Vs. Block Elements In HTML

Inline Elements

These elements take only the width which they need to be displayed, they do not occupy the entire width of the page. Also, when two or more inline elements are placed, they do not start from a new line, but rather adjacently.

Block Elements

These elements take the complete width of the page, irrespective of whether they need it or not. Also, when two or more block elements are placed, they occupy new lines.
