Extra Tags In HTML

Line Break

It is typically used to create line spaces (breaks) between paragraphs.

Two roads diverged in a yellow wood,
And sorry I could not travel both
And be one traveler, long I stood
And looked down one as far as I could
To where it bent in the undergrowth;

The poem displayed above has been embedded into only one paragraph tag.

Bold, Italic And Underline

As the name suggests, these tags are used for representing text in bold, italic and underlined.

This text is bold.
This text is italic.
This text is underlined.

Horizontal Rule

This tag simply adds a horizontal line in the page, used to seperate sections.

This is the first section.

This is the second section.
